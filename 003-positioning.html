Positioning

The positioning of html page elements can be accomplished by using a wide array of CSS techniques.

These techniques allow us to position the elements in relation to each other and with a variant distance.

In addition, CSS also offers the flexibility of positioning elements as far or as close to one another and in some cases, on top or underneath each other.

Different programmers use different techniques to positione elements around the page.

The properties that are used to accomplish these goals are:

  • The floats of the elements and in combination to the element's width:
     div {float: left} 
  • The z-index of the element in order to accomplish elevation
     span {z-index: 1;} 
  • The display of the elements:
     article {display: table-cell;} 
  • The position of the element which can be relative to the parent element or in a free format:
     span {position: absolute;} 

How was CSS used

  • We used the  display: table  option to properly position the main sections of the page
  • We used a position: absolute to be able to render the number in a place where it renders on top of other elements and is not confined by the parent tags
  • We used a display: block for the footer of the page since we wanted to keep the footer in one line by itself
