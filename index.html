CSS Tutorial

Introduction

Hello and welcome to the CSS Tutorial page.

This is a basic tutorial about CSS and how we can use differet css settings to accomplish functionality and a visibly pleasing content.

Before we begin with our CSS tutorial, we added some basic html elements in the page. These elements are some of the most common ones found in html pages.

We are assuming that you already have some familiarity with html tags and how they are nested within each other.

In this page, we have added some basic headers, nav tags, links, paragraphs and general structural tags.

At this point, this page is completely stripped of CSS and the only thing that renders is the lines of text that are being typed within the html document.

As we progress through this tutorial, we will be adding new CSS rules and we will be moving things around in order to make our page more pleasing to view and more functional.

What was added in the CSS code

Even though there is no CSS applied to the page, you can see that some formatting is already applied to it.

  2. The fonts of the page being in a specific preset font format
  3. Some intentation between the text and the window's frame
  4. Spacing between the different paragraphs and the headers of the page

As you progress through the pages of this tutorial, you will be able to see the css code that we added by viewing the things entered in the <code> tag.

For example, you will see code displaying like this:
